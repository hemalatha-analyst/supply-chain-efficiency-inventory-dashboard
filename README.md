📊 Supply Chain Efficiency & Inventory Optimization Dashboard
📌 Project Overview

This Power BI project is designed to analyze and optimize supply chain operations, inventory management, supplier performance, and demand forecasting.

It provides a single interactive dashboard for business teams to monitor inventory levels, reduce stockouts, improve delivery efficiency, and make data-driven decisions.

🎯 Objective

To build an interactive analytics dashboard that helps:

Monitor inventory levels in real time
Track supplier and warehouse performance
Reduce stockouts and overstock situations
Improve demand forecasting accuracy
Optimize supply chain efficiency
🏢 Business Importance

In real-world companies, supply chain inefficiencies lead to:

Revenue loss due to stockouts
High storage and holding costs
Delivery delays
Poor customer satisfaction

This dashboard helps solve these problems using data-driven insights.

🛠 Tools Used
Power BI Desktop
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)
📂 Dataset Description

The dataset includes:

Product details (ID, Name, Category)
Supplier and warehouse information
Inventory levels (opening, incoming, closing stock)
Sales and demand data
Delivery performance
Forecast accuracy metrics
⚙️ Power Query Transformations
Removed null values
Standardized column names
Changed data types (Date, Number, Text)
Created Month, Quarter, Year columns
Removed duplicates
Cleaned supplier and product names
📊 DAX Measures Used
Total Inventory
Total Units Sold
Stockout Count
Reorder Count
Average Lead Time
On-Time Delivery %
Forecast Accuracy %
Inventory Turnover Ratio
Inventory Holding Cost
📈 Dashboard Features
KPI Cards (Inventory, Stockouts, Delivery %, Forecast Accuracy)
Inventory by Product Category (Column Chart)
Supplier Delivery Performance (Bar Chart)
Monthly Sales Trend (Line Chart)
Delivery Status Distribution (Donut Chart)
Forecast vs Actual Demand (Clustered Column Chart)
Detailed Product Matrix Table
Interactive Filters (Region, Supplier, Category, Warehouse, Month)
🔍 Key Business Insights
Identifies products at high risk of stockouts
Detects suppliers causing delivery delays
Tracks inventory efficiency by category
Compares actual demand vs forecast accuracy
Highlights high and low-performing warehouses
Helps reduce inventory holding costs
